# RESPONSIVE IMAGE

The goal of responsive image is to serve the right image to the right screen size and device, in order to avoid downloading unnecesarry large images on smaller screens.

## WHEN TO USE RESPONSIVE IMAGES: THE 3 USESCASE

- Resoluation switching
- Desity Switching
- Art direction
