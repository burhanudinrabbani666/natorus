# HOW CSS RENDER WEBSITE

## THE VISUAL FORMATING MODEL

Algorithm that calculates boxes and determines the layout of these boxes, for each element in the render tree, in order to determine the final layout of the page.

- Dimensions of boxes: the box model
- Box type: inline, block and inline block
- Postioning scheme: floats and positioning
- Stacking contexts
- Others elemets in the order tree
- Viewport size, dimenisions of the images etc;

### THE BOXES MODEL

- **Content**: text, images etc
- **Padding**: transparent ares around the content, inside the box
- **Border**: goes around the padding and the content
- **Margin**: space between boxes
- **Fill area**: area thet gets filled with background color or background image
